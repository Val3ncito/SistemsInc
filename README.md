Proyecto: Sistems Inc

Descripción: Página Web orientada a la visualización de proyectos de construcción

Herramientas: HTML, JavScript, CSS, PHP

Link: https://sistemsinc.com/

No hace falta ningúna explicación inmersiva, la página de por si es bastante intuitiva, con solo ingresar ya pueden navegar de forma libre sin la necesidad de autenticarse pero acá va una pequeña guia:
